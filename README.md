# 石梅总演讲稿生成器 V6.0

## 🎯 项目简介

这是一个专为石梅（腾讯教育高管）量身定制的AI演讲稿生成系统，基于RAG（检索增强生成）技术和多篇真实演讲稿深度学习其演讲风格特征。系统采用三栏式现代化界面设计，支持智能数据增强，确保生成内容的真实性和专业性。

## ✨ 核心特性

### 🎨 现代化三栏界面
- **左侧导航栏**：动态AI图标、粉紫渐变主题、快捷导航
- **中间内容区**：演讲配置表单、实时生成结果
- **右侧面板**：系统概览、历史记录、统计信息

### 🧠 RAG智能数据增强
- **本地知识库**：内置腾讯教育真实项目数据
- **智能检索**：根据演讲主题自动匹配相关政策、案例和数据
- **真实性保障**：所有引用数据均来自真实项目，禁止胡编乱造

### 📚 知识库系统
- **4篇历史演讲稿**：深度学习石梅总演讲风格
  - 2026腾讯云合作伙伴大会教育专场致辞稿
  - 腾讯云&上海海事大学AI助力海事教育发布会致辞
  - 同济大学学术研讨会致辞稿
  - 其他高规格场合演讲稿
- **实时预览**：生成前展示已检索到的相关数据

### 🎯 多场景适配
- **目标听众**：管理层、技术团队、客户、员工、公众、学生、合作伙伴
- **演讲类型**：主题演讲、工作汇报、会议发言、培训讲座、庆典致辞
- **演讲风格**：专业正式、亲和友好、激励鼓舞、幽默风趣、学术严谨

### 🤖 AI智能生成
- **GLM-4 Plus集成**：调用智谱AI大模型生成高质量内容
- **风格深度学习**：
  - 开场风格：多场景适应的致敬方式
  - 核心理念："用户为本，科技向善"、"用数字科技，让教育更富创造力"
  - 价值主张："科技助教、连接兴学、专业为育"

## 📖 使用指南

### 1. 配置演讲参数
- **API密钥**：输入GLM-4 API密钥

- **演讲类型**：选择适合的场合类型

- 点击左侧"知识库"导航
- 查看4篇历史演讲稿全文

## 🔧 开发说明

### RAG引擎工作原理
1. **初始化**：加载 `education-data.json` 知识库
2. **检索**：根据演讲主题关键词匹配相关政策、案例
3. **增强**：将检索到的数据注入Prompt
4. **生成**：调用GLM-4生成基于真实数据的演讲稿

### 添加新知识
编辑 `knowledge-base/education-data.json`：
```json
{
  "cases": [
    {
      "id": "case-new",
      "title": "新案例标题",
      "date": "2025-XX",
      "partner": "合作方",
      "content": "案例内容",
      "results": "成果数据"
    }
  ]
}
```

## 🚨 常见问题

### Q: RAG数据检索失败？
A: 检查浏览器控制台网络请求，确保 `education-data.json` 可正常加载。

### Q: API连接失败？
A: 确认GLM API密钥有效，网络可访问智谱AI服务。

### Q: 生成内容不准确？
A: 确保启用RAG功能，系统会自动注入真实数据约束。
